# Developers Section

## Requirements:

- Python 3.10+

## Setup:

How to set up the bot:

1.  Cloning the repository

    ```bash
    git clone https://github.com/taporsnap37/miasma-public-bot; cd miasma-public-bot
    ```

2.  Running the bot
    **windows:**

    ```bash
    .\run.bat
    ```

    **linux:**

    ```bash
    chmod +x run.sh setup.sh
    ./run.sh
    ```
